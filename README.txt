Setup Instructions:
- In pgAdmin 4 crewate a new database named 3005_Assignment_3
- Go to the query tool for this database, and import the studentDatabaseCreation.sql file found in the gitHub repository
- Run the query
- Ensure that you have python and psycopg3 installed
- Edit the python file to change the password used in the connections to your pgAdmin 4 password
- Go to python IDLE and open then run the COMP3005_Assignment_3_Part_1.py file found in the gitHub repository
- Follow the program instructions once it runs
